@media запросы в CSS используются для создания адаптивного дизайна, который позволяет веб-страницам изменять своё представление в зависимости от характеристик устройства, на котором они отображаются. Это позволяет обеспечить оптимальный пользовательский интерфейс на различных экранах, таких как мобильные телефоны, планшеты и десктопы.

@media запросы позволяют применять CSS стили только тогда, когда определенные условия выполняются. Основные условия могут включать ширину экрана, высоту экрана, ориентацию (пейзажная или портретная), разрешение и другие параметры.

@media условие {
  /* CSS стили */
}

Стили, указанные в @media запросах, могут переопределять стили по умолчанию в зависимости от условий. Более специфичные @media запросы (с более узкими условиями) должны быть указаны после более общих.

Использование @media запросов для адаптивного дизайна:

Mobile-first:
/* Стили для мобильных устройств (по умолчанию) */
body {
  font-size: 14px;
}

/* Стили для экранов шириной более 768px */
@media (min-width: 768px) {
  body {
    font-size: 16px;
  }
}

/* Стили для экранов шириной более 1024px */
@media (min-width: 1024px) {
  body {
    font-size: 18px;
  }
}

Desktop-first подход:
/* Стили для больших экранов (по умолчанию) */
body {
  font-size: 18px;
}

/* Стили для экранов шириной до 1024px */
@media (max-width: 1024px) {
  body {
    font-size: 16px;
  }
}

/* Стили для экранов шириной до 768px */
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}
