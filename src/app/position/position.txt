1. position:static
Свойство position: static является значением по умолчанию для всех элементов в CSS.

- Не поддается управлению через top, right, bottom, left;
- Элемент с position: static не выходит за рамки своего естественного места в потоке документа, что делает его неподвижным относительно других элементов;
-Не создает новый контекст наложения (В отличие от других типов позиционирования (например, relative, absolute или fixed), static не меняет контекст наложения элементов.)

2. position: relative
Свойство position: relative в CSS позволяет позиционировать элемент относительно его начального положения в обычном потоке документа. При использовании этого свойства элемент остается на своем месте в потоке, но его можно смещать относительно начального местоположения с помощью свойств top, right, bottom, и left.
Не будут работать одновременно указанные top и bottom, left и right. Нужно использовать только одну границу из каждой пары.

- Элемент остается в потоке документа (Даже если элемент смещен, он сохраняет свое место в общем потоке документа. Это означает, что другие элементы по-прежнему ведут себя так, как будто элемент не был смещен.);
- Смещение относительно начальной позиции(Можно использовать свойства top, right, bottom, и left для перемещения элемента относительно его нормального положения. Эти свойства изменяют его видимое расположение, но не влияют на положение соседних элементов)
- Создает новый контекст для абсолютно позиционированных дочерних элементов(Если у родительского элемента установлено position: relative, дочерние элементы с position: absolute будут позиционироваться относительно его границ, а не всего документа);

Элемент можно смещать, не нарушая поток документа.
position: relative полезен для создания локальных контекстов для абсолютного позиционирования.
Работа с наложением (z-index) для управления видимостью перекрывающихся элементов.

(Вопрос 3):
Использование position: relative позволяет создавать эффекты сдвига элементов в пределах их нормального потока, не изменяя их место в обычном потоке документа. Этот метод полезен для добавления небольших изменений в положение элементов, сохраняя их исходное место для других элементов.
Этот метод удобен для внесения небольших изменений в положение элементов, сохраняя их интеграцию в нормальный поток документа.

3. position: absolute
Свойство position: absolute в CSS позволяет полностью извлечь элемент из нормального потока документа и позиционировать его относительно ближайшего предка, у которого установлено свойство position, отличное от static. Если такого предка нет, то элемент будет позиционироваться относительно области просмотра (viewport) страницы.

- Выходит из нормального потока(Элемент с position: absolute удаляется из обычного потока документа. Это значит, что он не влияет на расположение соседних элементов, и те ведут себя так, будто этого элемента нет)
- Позиционируется относительно ближайшего предка с position(Если предок имеет position: relative, absolute, fixed, или sticky, то элемент с position: absolute будет позиционироваться относительно него)
- Использует свойства top, right, bottom, и left для точного позиционирования(Эти свойства позволяют задавать отступы относительно краев контейнера, к которому привязан элемент)
-Контекст наложения (z-index)(Элементы с position: absolute могут перекрывать другие элементы. Значение z-index помогает контролировать, какой элемент будет "выше" при наложении)

4. position: fixed
position: fixed — это один из типов позиционирования в CSS, который позволяет элементу оставаться закреплённым в одном месте на экране, независимо от прокрутки страницы. Элемент с position: fixed позиционируется относительно окна браузера (viewport) и не изменяет своё положение, когда пользователь прокручивает страницу.

- Относительно окна браузера (viewport): Элемент фиксируется относительно окна, а не относительно документа или других элементов.
- Не зависит от прокрутки: Позиция элемента остаётся постоянной, даже если пользователь прокручивает страницу.
- Изъят из обычного потока документа: Элемент с position: fixed не влияет на соседние элементы, и они не учитывают его в расположении.

5. position: sticky
position: sticky — это позиционирование, которое сочетает в себе особенности двух других типов: relative и fixed. Элемент с position: sticky ведет себя как элемент с position: relative до тех пор, пока не достигает определенной позиции при прокрутке страницы, после чего он "прилипает" к этой позиции и ведет себя как элемент с position: fixed

- Относительно потока документа: Элемент с position: sticky изначально находится в нормальном потоке документа и позиционируется как элемент с relative
- Фиксация при прокрутке: Когда элемент достигает заданной позиции (например, top: 0), он "прилипает" к ней и начинает вести себя как элемент с position: fixed
- Прекращает фиксацию: Элемент перестает быть "приклеенным", когда его родительский элемент заканчивается (то есть элемент не может выйти за пределы своего контейнера)
