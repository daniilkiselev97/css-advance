1. position:static
Свойство position: static является значением по умолчанию для всех элементов в CSS.

- Не поддается управлению через top, right, bottom, left;
- Элемент с position: static не выходит за рамки своего естественного места в потоке документа, что делает его неподвижным относительно других элементов;
-Не создает новый контекст наложения (В отличие от других типов позиционирования (например, relative, absolute или fixed), static не меняет контекст наложения элементов.)

2. position: relative
Свойство position: relative в CSS позволяет позиционировать элемент относительно его начального положения в обычном потоке документа. При использовании этого свойства элемент остается на своем месте в потоке, но его можно смещать относительно начального местоположения с помощью свойств top, right, bottom, и left.
Не будут работать одновременно указанные top и bottom, left и right. Нужно использовать только одну границу из каждой пары.

- Элемент остается в потоке документа (Даже если элемент смещен, он сохраняет свое место в общем потоке документа. Это означает, что другие элементы по-прежнему ведут себя так, как будто элемент не был смещен.);
- Смещение относительно начальной позиции(Можно использовать свойства top, right, bottom, и left для перемещения элемента относительно его нормального положения. Эти свойства изменяют его видимое расположение, но не влияют на положение соседних элементов)
- Создает новый контекст для абсолютно позиционированных дочерних элементов(Если у родительского элемента установлено position: relative, дочерние элементы с position: absolute будут позиционироваться относительно его границ, а не всего документа);

Элемент можно смещать, не нарушая поток документа.
position: relative полезен для создания локальных контекстов для абсолютного позиционирования.
Работа с наложением (z-index) для управления видимостью перекрывающихся элементов.
