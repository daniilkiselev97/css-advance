Атрибуты data-* в HTML используются для хранения пользовательских данных, которые могут быть полезны для JavaScript и CSS. Эти атрибуты позволяют добавлять дополнительные метаданные к элементам HTML без изменения их структуры или внешнего вида.

Преимущества и использование:
- Хранение дополнительной информации: Атрибуты data-* позволяют хранить информацию, которую нельзя выразить с помощью стандартных атрибутов HTML, без риска конфликтов с будущими стандартами.

- Удобство для JavaScript: Они позволяют хранить данные, которые могут быть использованы в JavaScript, упрощая доступ и манипуляцию этими данными.

- Избежание загрязнения HTML: Использование data-* атрибутов предотвращает загрязнение HTML дополнительными пользовательскими атрибутами, которые могут конфликтовать с существующими стандартными атрибутами.

- Кросс-браузерная поддержка: Атрибуты data-* поддерживаются всеми современными браузерами.
